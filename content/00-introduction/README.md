# Chapter 0: Introduction & Setup

Welcome to the course. This first chapter sets the stage: what you'll build, what you'll need, and how everything fits together.

## Lessons

1. **[What you'll build](./01-what-youll-build.md)** — The end state, with a video-walkthrough mental model

## Time estimate

~45 minutes total.

## What you'll know by the end

- A clear mental model of what a coding agent harness contains
- Why pi-mono (and every other coding agent) splits into transport, agent, UI, and product layers
- A working dev environment for building your own harness in subsequent chapters

## Why this chapter matters

If you skip the foundations and dive into code, you'll find yourself confused by *why* things are structured the way they are. The biggest insight in this entire course is in Lesson 0.2 — the layering pattern. Don't skip it.
