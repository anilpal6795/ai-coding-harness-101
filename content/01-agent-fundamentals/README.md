# Chapter 1: Agent Fundamentals

Before writing any code, we need to be precise about *what an agent is*. The word is used loosely in the industry — let's pin it down.

## Lessons

1. **[From chatbot to agent](./01-chatbot-to-agent.md)** — The single conceptual jump
2. **[The agentic loop](./02-the-agentic-loop.md)** — The pattern at the core of every agent
3. **[What makes a coding agent](./03-what-makes-a-coding-agent.md)** — Why coding agents are a particular subgenre
4. **[Anatomy of pi](./04-anatomy-of-pi.md)** — A guided tour of the reference implementation

## Time estimate

~60 minutes total.

## What you'll know by the end

- The precise definition of an agent (and why "AI agent" the marketing term is mostly meaningless)
- Why the agentic loop is universal — every agent is some variation of it
- What makes coding agents different from generic agents
- A concrete map of pi-mono so you can find anything quickly

## Why this chapter matters

You're going to build an agent. If you don't know what an agent is, you'll cargo-cult features from other agents without understanding why they exist. This chapter ensures you know **why** every line of code in the next 9 chapters exists.
